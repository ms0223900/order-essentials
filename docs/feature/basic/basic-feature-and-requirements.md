### **一頁式購物網站基本功能與實現目標**

| **功能類別**   | **基本功能**                                                                 | **實現目標**                         |
| -------------- | ---------------------------------------------------------------------------- | ------------------------------------ |
| **商品展示**   | - 商品圖片輪播或展示區<br>- 商品名稱、價格、簡介<br>- 庫存狀態顯示           | 清晰展示商品資訊，吸引用戶購買意願。 |
| **購物車**     | - 加入購物車按鈕<br>- 購物車即時預覽（數量、總價）<br>- 編輯購物車功能       | 讓用戶輕鬆管理選購商品，提升轉換率。 |
| **結帳流程**   | - 一頁式結帳（填寫配送、付款資訊）<br>- 多種付款方式（信用卡、第三方支付等） | 簡化結帳步驟，減少用戶流失。         |
| **響應式設計** | - 自動適應手機、平板、電腦螢幕                                               | 確保所有裝置用戶體驗一致。           |

---

### **功能清單**

商品展示區：
  - 高品質圖片或影片展示。
  - 簡潔的商品描述與價格標示。
  - 庫存狀態（如「僅剩 3 件」）。
購物車：
  - 一鍵加入購物車。
  - 結帳表單自動填入常用資訊（如記憶配送地址）。
  - 支援多種付款方式（信用卡、Apple Pay、LINE Pay 等）。
技術支援：
  - RWD（響應式網頁設計）。
  - 跨瀏覽器相容性測試（Chrome、Safari、Edge）。
